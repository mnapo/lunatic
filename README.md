## Lunatic:

"from the moon" or "moonstruck"

---

## Purpose (v0.0.2)

A modern AI ecosystem for Lua.

Lunatic is being redefined from a general scientific computing idea into a focused ecosystem for machine learning, neural networks, and AI tooling in Lua.

---

## Philosophy

> Readable first, performant by design

The system should remain understandable at every layer of abstraction.

---

## Note

This project is under active architectural design (V0.0.2 setting up phase).  
Interfaces and internal structure may evolve as the core system stabilizes.
